## Verification Files

If you need to verify ownership with Google, you can create a verification file:
1. In Google Search Console, choose "HTML file" verification method
2. Download the verification file (e.g., `google1234567890abcdef.html`)
3. Place it in the root directory of your site
4. Click verify
